One-time setup (2 minutes): Create a free Firebase project to enable cross-device real-time chat. Follow the steps below:
1
Go to console.firebase.google.com and sign in with Google
2
Click "Create a project" → Type any name → Click Continue → Disable Analytics → Create Project
3
In the left sidebar, click "Build" → "Realtime Database" → "Create Database"
4
Choose any location → Select "Start in test mode" → Click "Enable"
5
Click the ⚙️ gear icon (top-left) → "Project settings" → Scroll down → Click the </> (Web) icon
6
Enter any nickname → "Register app" → Copy the firebaseConfig = {...} part → Paste below
